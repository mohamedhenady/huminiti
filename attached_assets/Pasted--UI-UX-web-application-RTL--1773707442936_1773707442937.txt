أنت مصمم UI/UX محترف. مهمتك تصميم web application كامل باللغة العربية (RTL) 
لنظام إدارة دفعات الأدوية — تعاون بين صيدلية السعوي ومؤسسة هيومنيتي.

الطابع: احترافي طبي، نظيف، موثوق
اللغة: عربي 100%، اتجاه RTL كامل
الخطوط: Cairo أو Tajawal (Google Fonts)
الأجهزة: Responsive — Desktop أولوية + Mobile

- Background: #F4F6F9 (رمادي فاتح جداً)
- Card: #FFFFFF
- Text Primary: #2C3E50
- Text Secondary: #7F8C8D

🔐 صفحة تسجيل الدخول
━━━━━━━━━━━━━━━━━━━━━━━━━━

Layout: صفحة منقسمة نصين
- الجانب الأيسر (60%): صورة خلفية طبية داكنة + شعار صيدلية السعوي + 
  شعار هيومنيتي + جملة ترحيبية
- الجانب الأيمن (40%): الفورم

الفورم:
- عنوان: "تسجيل الدخول"
- حقل: اسم المستخدم (أيقونة شخص)
- حقل: كلمة المرور (أيقونة قفل + زرار إظهار/إخفاء)
- زرار: "دخول" — Primary color، عرض كامل، مستطيل بزوايا دائرية
- تنبيه خطأ يظهر تحت الفورم باللون الأحمر لو البيانات غلط
- الـ role يتحدد أوتوماتيك من السيرفر بناءً على بيانات الدخول

Mobile: الصفحة كلها الفورم فقط + شعارات صغيرة فوق

━━━━━━━━━━━━━━━━━━━━━━━━━━
🧭 الـ Layout العام بعد الدخول
━━━━━━━━━━━━━━━━━━━━━━━━━━

Sidebar (يمين — ثابت على Desktop):
- شعار صيدلية السعوي فوق
- قائمة التنقل:
  • 🏠 الدفعات
  • 💊 قاموس الأدوية (صيدلية فقط)
- اسم المستخدم + الـ role في الأسفل
- زرار تسجيل خروج

Top Bar:
- اسم الصفحة الحالية (Breadcrumb)
- اسم المستخدم + صورة رمزية

Mobile: Bottom Navigation Bar بدل الـ Sidebar
- أيقونات فقط مع label صغير

━━━━━━━━━━━━━━━━━━━━━━━━━━
🏠 صفحة الدفعات (الرئيسية)
━━━━━━━━━━━━━━━━━━━━━━━━━━

Header الصفحة:
- عنوان "الدفعات" (يمين)
- زرار "＋ دفعة جديدة" + زرار "📥 استيراد Excel" (يسار)
- كلا الزرارين ظاهرين للـ roleين

Stats Row (4 كاردات صغيرة في الأعلى):
- إجمالي الدفعات
- الدفعات الجارية
- الدفعات المكتملة
- إجمالي الأشخاص

قائمة الدفعات — Grid من الكاردات (3 في الصف Desktop، 1 Mobile):
كل كارت دفعة يحتوي:
- اسم الدفعة (Bold، كبير)
- Badge حالة: "جارية" (برتقالي) / "مكتملة" (أخضر)
- عدد الأشخاص الكلي
- شريط تقدم أفقي ملون:
  • النسبة المئوية للأشخاص اللي استلموا
  • مثال: "١٢ من ٢٠ استلموا" تحت الشريط
- إجمالي التكلفة قبل الخصم (رمادي، خط صغير)
- إجمالي التكلفة بعد الخصم (أزرق، Bold)
- زرار "عرض التفاصيل" في أسفل الكارت

━━━━━━━━━━━━━━━━━━━━━━━━━━
📦 صفحة تفاصيل الدفعة
━━━━━━━━━━━━━━━━━━━━━━━━━━

Breadcrumb: الدفعات ← اسم الدفعة

Header الصفحة:
- اسم الدفعة (H1)
- Badge الحالة
- صيدلية فقط: زرار "تعديل الدفعة"

Summary Row (3 كاردات):
- شريط التقدم الكلي للدفعة مع النسبة المئوية
- إجمالي التكلفة قبل الخصم
- إجمالي التكلفة بعد الخصم

فوق الجدول:
- بحث بالاسم أو رقم الفاتورة (يمين)
- فلتر: الكل / لم يستلم / استلم (Tabs أو Dropdown)
- صيدلية فقط: زرار "＋ إضافة شخص"

جدول الأشخاص:
الأعمدة: الاسم | الهاتف | العنوان | رقم الفاتورة | 
التكلفة قبل | التكلفة بعد | تقدم الأدوية | الاستلام | إجراءات

- تقدم الأدوية: 3 دوائر صغيرة ملونة 
  🔴 ordered | 🟠 prepared | 🟢 ready
  مع أرقام: مثال "٣ | ٢ | ٥"
- الاستلام: أيقونة ✅ خضراء أو ⏳ رمادية
- إجراءات (صيدلية فقط): زرار "تعديل" + زرار "عرض"

Mobile: الجدول يتحول لكاردات مكدسة

━━━━━━━━━━━━━━━━━━━━━━━━━━
👤 صفحة تفاصيل الشخص
━━━━━━━━━━━━━━━━━━━━━━━━━━

Breadcrumb: الدفعات ← اسم الدفعة ← اسم الشخص

Header الصفحة:
- اسم الشخص (H1) + رقم الهاتف (رمادي تحته)
- Badge استلام: "استلم ✅" أخضر / "لم يستلم ⏳" رمادي

Info Row (كاردات صغيرة):
- العنوان 📍
- رقم الفاتورة 🧾
- التكلفة قبل الخصم
- التكلفة بعد الخصم (أكبر + Bold + أزرق)

جدول الأدوية:
الأعمدة: اسم الدواء (عربي) | (إنجليزي) | النوع | السعر | بعد الخصم | الحالة

- النوع: Badge صغير "محلي -٢٠%" (أخضر فاتح) أو "مستورد -١٠%" (أزرق فاتح)
- الحالة: 3 أزرار Toggle متسلسلة:
  [طُلب] [جُهِّز] [جاهز]
  الزرار الحالي يكون Filled، الباقيين Outlined
  صيدلية فقط: الأزرار قابلة للضغط
  هيومنيتي: الأزرار Display فقط (disabled)

Footer الصفحة (صيدلية فقط):
- زرار "✅ تسجيل الاستلام" — Primary color، كبير، ممتد
- يظهر فقط لما كل الأدوية حالتها "جاهز"
- لو مش كل الأدوية جاهزة: الزرار رمادي disabled مع tooltip 
  "لازم كل الأدوية تكون جاهزة الأول"

━━━━━━━━━━━━━━━━━━━━━━━━━━
💊 صفحة قاموس الأدوية (صيدلية فقط)
━━━━━━━━━━━━━━━━━━━━━━━━━━

Header:
- عنوان "قاموس الأدوية"
- زرار "＋ إضافة دواء" (يسار)

Toolbar:
- بحث بالاسم (عربي أو إنجليزي)
- فلتر: الكل / محلي / مستورد

جدول الأدوية:
الأعمدة: الاسم العربي | الاسم الإنجليزي | النوع | السعر | إجراءات
- النوع: Badge "محلي" أخضر / "مستورد" أزرق
- السعر: بالجنيه المصري
- إجراءات: أيقونة تعديل ✏️ + أيقونة حذف 🗑️ (مع تأكيد قبل الحذف)

━━━━━━━━━━━━━━━━━━━━━━━━━━
📥 Modal استيراد Excel
━━━━━━━━━━━━━━━━━━━━━━━━━━

- Drag & Drop منطقة رفع الملف
- أو زرار "اختر ملف"
- بعد الرفع: preview جدول بالبيانات المستوردة
- زرار "تأكيد الاستيراد" + زرار "إلغاء"

🔔 عناصر UX عامة
━━━━━━━━━━━━━━━━━━━━━━━━━━

- Toast Notifications (يمين أسفل):
  • أخضر: "تم الحفظ بنجاح ✅"
  • أحمر: "حدث خطأ، حاول مرة أخرى ❌"
- Loading Skeleton على كل صفحة بيانات
- Empty State لما مفيش دفعات: أيقونة + نص + زرار إضافة
- Confirmation Dialog قبل أي حذف
- كل الأزرار والـ actions المتعلقة بالصيدلية 
  مخفية كامل عن موظف هيومنيتي (مش disabled — مش موجودة خالص)
